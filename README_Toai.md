# Project Buổi 5 - Authentication & User Management

Đây là dự án ứng dụng web hoàn chỉnh bao gồm các chức năng xác thực và quản lý người dùng, được xây dựng với MERN stack (MongoDB, Express, React, Node.js).

## Thành viên nhóm

- **Nguyễn Chí Toại**: Backend Developer & Git Manager
- **Hồng Phước Thịnh**: Frontend Developer

## Các chức năng chính

- Đăng ký, Đăng nhập, Đăng xuất.
- Quản lý thông tin cá nhân (Xem, Cập nhật tên/email, Đổi avatar).
- Phân quyền User và Admin.
- Trang Admin để xem và xóa người dùng.
- Quên mật khẩu và Đặt lại mật khẩu qua email (hoặc token test).
- Cập nhật ảnh đại diện (sử dụng Base64).

- MongoDB (có thể dùng Atlas hoặc cài đặt local)
